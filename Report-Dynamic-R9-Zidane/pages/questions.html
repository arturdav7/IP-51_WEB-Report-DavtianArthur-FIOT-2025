Контрольні питання

  1. Яка структура HTML-документу

    Стандартна структура: <!DOCTYPE html>, далі кореневий тег <html>, всередині нього два основні розділи: <head> (метадані сторінки: кодування, заголовок, стилі, скрипти) та <body> (усі видимі елементи сторінки: заголовки, текст, зображення, форми, таблиці тощо).

  2. Дати визначення тега

    Тег — це елемент розмітки HTML, який описує структуру або зміст сторінки, наприклад <p>, <h1>, <img>. Атрибут — це додаткова характеристика тегу у форматі імʼя="значення" (наприклад class="btn", src="image.png", alt="Опис"), яка уточнює його поведінку або вигляд.

  3. Які теги використовуються при роботі з таблицями

    Основні теги таблиць: <table> — контейнер таблиці, <caption> — заголовок таблиці, <thead> — рядки шапки, <tbody> — основні дані, <tfoot> — підсумковий блок, <tr> — рядок, <th> — комірка-заголовок, <td> — комірка з даними.

  4. Які теги використовуються при роботі з зображеннями

    Базовий тег — <img> (виведення зображення, основні атрибути: src, alt, width, height). Для групування зображення з підписом використовують <figure> та <figcaption>, а для адаптивних зображень — <picture> та <source>.

  5. Які теги використовуються при роботі зі списками

    Маркований список — <ul>, нумерований — <ol>, елементи списку — <li>. Для описових списків використовують <dl> (description list), <dt> (термін) та <dd> (опис терміну).

  6. Перерахувати засоби опису таблиць в HTML.

    Для створення таблиць HTML використовує такі елементи:

    • <table> — основний контейнер таблиці;
    • <caption> — заголовок таблиці;
    • <thead> — шапка таблиці (заголовки стовпців);
    • <tbody> — основна частина таблиці з даними;
    • <tfoot> — підсумкова частина таблиці;
    • <tr> — рядок таблиці;
    • <th> — комірка-заголовок рядка або стовпця;
    • <td> — комірка з даними;
    • colspan і rowspan — атрибути, що об’єднують комірки;
    • border, cellpadding, cellspacing — атрибути для застарілого форматування (сьогодні замінені CSS);
    • CSS-властивості (border, padding, text-align, background тощо) — для стилізації таблиць.
